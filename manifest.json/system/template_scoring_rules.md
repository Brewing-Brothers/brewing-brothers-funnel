# Template Intelligence Scoring Rules
## Brewing Brothers — Website Creation Agent

---

## Core Principle
The goal is NOT the most beautiful template.
The goal is the most **conversion-efficient structure** for a one-page funnel.

Priority order:
1. Conversion clarity
2. Funnel flow
3. Mobile usability
4. Editing simplicity
5. Brand fit

---

## Required Funnel Elements
Templates should ideally contain:
- Hero section with CTA
- Problem / solution explanation
- Offer stack
- Trust section
- FAQ section
- Final CTA

Missing multiple elements lowers score significantly.

---

## Scoring Scale
| Score | Meaning |
|-------|---------|
| 5 | Excellent |
| 4 | Strong |
| 3 | Acceptable |
| 2 | Weak |
| 1 | Poor |

---

## Template Selection Rule
1. Score all templates using template_scorecard.csv
2. Select top 10
3. Analyze those deeply
4. Extract patterns → update patterns/ and snippets/
5. Ignore the rest until Phase 5

---

## What NOT to Extract
Do NOT extract:
- Color palettes
- Fonts
- Background images
- Branding styles

These are cosmetic. Extract **conversion architecture**.

---

## Template Rejection Criteria
Reject templates that have:
- No clear CTA
- Multi-page navigation required
- Heavy animations that distract from conversion
- Checkout hidden or unclear
- Poor mobile layout

---

## System Discipline
- Never analyze 100+ templates simultaneously
- Top 10 first, always
- Score → Analyze → Extract → Store → Build
